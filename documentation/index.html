A. Welcome to Insurance Template Documentation!

Thank you for purchase Insurance HTML5 Template!

I hope this template will be helpful for you - if you'll have any problems with using it, please contact me via this support email. Feel free to send me your website URL, which has been created using this theme - I'll be happy!

B. Template installation

This template has been written in HTML5, CSS3 and JavaScript (with jQuery). After downloading this template from your user panel on ThemeForest, you have to unzipped all files to some local folder on your computer.

All of the template files are placed in "insurance" directory. Select all template files and open your FTP Client (for example FileZilla). Then, make connection with your server (you may need your server name, FTP username and password).

Locate the domain folder, where you want to install this template and then open "public_html" (or similar) directory. After that, make an upload of all files to selected folder - that's all!

C. HTML structure

In "insurance" directory you can find eleven template files:

  • index.html
    this is home page of this template

  • agents.html
    this is example page for agents, with example usage of progress bars shortcode

  • blog.html
    this is main blog page

  • blog-single.html
    this is single blog post page

  • car-insurance.html,
    travel-insurance.html,
    house-insurance.html,
    life-insurance.html
    these pages are an example of insurance promo, with special place for insurance parameters

  • claims.html
    this is page for insurance claims

  • contact.html
    this is separate contact page with Google Map and contact form, if you don't want to use contact popup

  • 404.html
    this is page for 404 error (page not found)

D. CSS structure

Stylesheets are located in "_assets/_css" directory. For all .html files you need to include only one .css file - "style.css". This template has been written using CSS3 - if you haven't used it before, go to this w3schools.com website - it's very helpful introduction.

Open "style.css" file (this is main stylesheet for Insurance) to see the CSS structure.

E. JavaScript

Insurance HTML5 template has only one javascript file, located into "_assets/_js/" directory - functions.js; This is main javascript file with general functions and settings.

You shouldn't make any configuration changes in "functions.js" file - any customizations, like Google Map marker position or additional fields in Quote Popup should be done in HTML file.
